📂 2.1 · Grundlagen der Terminalbedienung

Öffnen Sie das Terminal z. B. mit Strg+Alt+T oder über das Anwendungsmenü. Es öffnet sich ein Fenster mit einem Cursor, welcher die aktuelle Eingabeposition markiert.
Die Bedienung des Terminals erfolgt in der Regel ausschließlich per Tastatur. In die Eingabezeile können verschiende Befehle eingetippt werden. Sobald man die Enter-Taste Enter drückt wird der eingegebene Befehl ausgeführt und dessen Ausgabe im Terminal angezeigt.

Das Terminal arbeitet dabei stets in einem sogenannten Arbeitsverzeichnis (ein Verzeichnis im Dateisystem des Rechners). Startet man ein neues Terminal befindet man sich meist im persönlichen Verzeichnis des angemeldeten Benutzers /home/benutzername.

hsg@hsg-HP-ProBook-650-G4: ~
hsg@hsg-HP-ProBook-650-G4:~$ pwd
/home/hsg

Durch entsprechende Befehle kann das Arbeitsverzeichnis jederzeit gewechselt werden.

Nachfolgend werden einige der wichtigsten Befehle erklärt:

🔧 Befehl📋 Funktion
pwdZeigt den Pfad des aktuellen Arbeitsverzeichnisses an
ls -lhListet den Inhalt des aktuellen Verzeichnisses auf (detailliert)
cd xxxWechselt in das Verzeichnis xxx
cd ~Wechselt ins Home-Verzeichnis
cd ..Wechselt ins übergeordnete Verzeichnis
mkdir xxxErstellt ein neues Verzeichnis xxx
rmdir xxxLöscht ein (leeres!) Verzeichnis xxx
cp xxx yyyKopiert die Datei xxx nach yyy
echo "Hallo"Gibt den Text Hallo im Terminal aus
echo "Hallo" > xxxSchreibt Hallo in die Datei xxx (überschreibt!)
cat xxxGibt den Inhalt der Datei xxx im Terminal aus
ARBEITSAUFTRAG 1 Dateisystemstruktur erkunden

Verwenden Sie die Befehle cd ls -lh pwd und cat um sich einen Überblick über die Dateisystemstruktur zu verschaffen.

a)

Geben Sie die Namen aller Verzeichnisse direkt unterhalb der Wurzelebene (Pfad '/') an.

b)

Die Verzeichnisse aus a) haben größtenteils "sprechende Namen", d. h. in ihren Namen verbirgt sich ein Verwendungszweck bzw. die Art der enthaltenen Dateien. Stellen Sie für mindestens drei der Verzeichnisse eine Hypothese auf, welchem Zweck das jeweilige Verzeichnis dient.

c)

Legen Sie in Ihrem Benutzerverzeichnis eine sinnvolle Verzeichnisstruktur zum Organisieren von Schulmaterialien an.

d)

Installieren Sie das Programm tree mit sudo apt-get install tree.
Wechseln Sie anschließend direkt in Ihr Benutzerverzeichnis und führen Sie den Befehl tree aus. Beschreiben Sie, welche Funktionalität dieser Befehl bietet.

📝 2.2 · Textdateien und Programmcode im Terminal bearbeiten

Im Terminal können auch Programme ausgeführt werden.

Mit dem Befehl nano beispiel.txt kann beispielsweise ein interaktiver Texteditor gestartet und darin die Datei beispiel.txt (aus dem aktuellen Arbeitsverzeichnis) geöffnet werden.

Die Bedienung erfolgt ebenfalls allein per Tastatur. Der gewünschte Text kann an der aktuellen Cursorposition direkt eingetippt werden.
Wichtige Shortcuts sind am unteren Rand eingeblendet:

GNU nano – beispiel.txt
^G Hilfe    ^O Speichern    ^W Wo ist    ^K Ausschneiden    ^T Ausführen
^X Beenden    ^R Öffnen    ^\ Ersetzen    ^U Einfügen    ^J Ausrichten

Dabei steht die Darstellung ^X zum Beenden des Editors für Strg+X, während zum Speichern der aktuellen Datei Strg+O gedrückt werden muss.

Auch eigene Java- oder Python-Programme können im Terminal ausgeführt werden. Während Python standardmäßig integriert ist, muss für Java zunächst das JDK installiert werden:

Java installieren
$ sudo apt update    # Prüfe, ob Updates vorhanden sind
$ sudo apt upgrade    # Installiere Updates ggf.; mit J bestätigen, um fortzufahren
$ sudo apt install openjdk-11-jdk    # openjdk für Java herunterladen und installieren
$ java -version    # aktuelle Java-Version anzeigen
$ javac Dateiname.java  # Kompilieren
$ java Dateiname        # Ausführen
ARBEITSAUFTRAG 2 Java-Programm im Terminal schreiben

Verwenden Sie ausschließlich das Terminal, um ein kleines Java-Programm zu schreiben und anschließend zu testen.

a)

Nutzen Sie den Texteditor nano um eine Datei Testprogramm.java im Benutzerverzeichnis anzulegen und zu bearbeiten.

b)

Recherchieren Sie (falls notwendig), was man unter sogenannten Kommandozeilenparametern versteht und wie diese in Java eingelesen werden können.
Schreiben Sie ein Programm, das einen beliebigen Text als Kommandozeilenparameter erhält und diesen in umgekehrter Buchstabenreihenfolge wieder ausgibt.

💡
Hinweis zu Kommandozeilenparametern:
Alles, was nach dem Programmnamen beim Aufruf steht, landet im Array args[] der main-Methode. Mit args[0] greifst du auf den ersten Parameter zu.
c)

Speichern und testen Sie das Programm aus b).
